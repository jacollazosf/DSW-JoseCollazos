AUTOR

Aside

La <aside> etiqueta define algún contenido además del contenido en el que se coloca.
El contenido aparte debe estar relacionado indirectamente con el contenido circundante Sugerencia: el <aside> contenido a menudo se coloca como una barra lateral en un documento.
Nota: El <aside> elemento no se representa como algo especial en un navegador. Sin embargo, puede usar CSS para diseñar el <aside> elemento.
